
우아한테크코스 웹 백엔드 최종 코딩테스트 대비를 위한 연습용 프로젝트: 다리 건너기

---

## 🎯 프로젝트 개요

우아한테크코스 최종 코딩테스트를 대비하기 위한 연습용 프로젝트입니다.   
다리 건너기 프로그램을 구현합니다.

---

## 📝 구현 기능 목록

### 1. 게임 시작
- [ ] **시작 문구 출력**
- [ ] **다리 길이 입력**
  - 입력 안내 문구 출력
  - 다리 길이 입력
  - **다리 길이 예외 처리**
    - 빈값 / 공백 불가
    - 3 이상 20 이하의 숫자
  - 예외 발생 시 에러 문구 출력, 재입력
- [ ] **다리 생성**
    - 건널 수 있는 칸 선정: 0과 1 중 무작위 값을 이용
    - 위 칸을 건널 수 있는 경우 U, 아래 칸을 건널 수 있는 경우 D값으로 표현
    - 무작위 값이 0인 경우 아래 칸, 1인 경우 위 칸이 건널 수 있는 칸

### 2. 게임 진행
- [ ] **이동할 칸 입력**
    - 입력 안내 문구 출력
    - 이동할 칸 입력
    - **이동할 칸 예외 처리**
        - 빈값 / 공백 불가
        - U 또는 D
    - 예외 발생 시 에러 문구 출력, 재입력
- [ ] **칸 이동 시도**
    - 위 칸은 대문자 U, 아래 칸은 대문자 D
    - 이동한 칸을 건널 수 있다면 O, 건널 수 없다면 X로 표시
    - 다리를 끝까지 건너면 게임 종료 프로세스 시작
- [ ] **칸 이동 성공**
    - 게임 진행 프로세스 반복
    - 다리를 끝까지 건너면 게임 종료 프로세스 시작
- [ ] **칸 이동 실패**
    - 게임 실패 프로세스 시작

### 3. 게임 실패
- [ ] **게임 재시도 여부 입력**
    - 입력 안내 문구 출력
    - 게임 재시도 여부 입력
    - **게임 재시도 여부 예외 처리**
        - 빈값 / 공백 불가
        - R 또는 Q
    - 예외 발생 시 에러 문구 출력, 재입력
- [ ] **게임 재시도**
    - 다리는 재사용
    - 총 시도한 횟수는 첫 시도를 포함해 게임을 종료할 때까지 시도한 횟수임
    - 게임 진행 프로세스 시작

### 4. 게임 종료
- [ ] **최종 게임 결과 출력**
- [ ] **게임 성공 여부 출력**
- [ ] **총 시도 횟수 출력**
